大型家具回收預約平台 - 資料庫設計

1. `users` 表格

欄位名稱 描述
user_id 用戶唯一識別碼 (Primary Key)
name 用戶姓名
phone 用戶聯絡電話
email 用戶電子郵件
password 用戶密碼
created_at 帳戶創建時間
updated_at 帳戶資料更新時間

2. `furniture_requests` 表格

欄位名稱 描述
request_id 回收申請唯一識別碼 (Primary Key)
user_id 用戶 ID (Foreign Key, 來自 `users`)
point_id 回收點 ID (Foreign Key, 來自 `recycling_points`)
slot_id 服務時段 ID (Foreign Key, 來自 `service_slots`)
item_type 家具種類 (如床墊、沙發等)
quantity 家具數量
size 家具尺寸 (長 x 寬 x 高)
photo_url 家具照片的 URL
status 申請狀態 (如 NEW, ACCEPTED, CANCELED 等)
created_at 回收申請創建時間

3. `recycling_points` 表格

欄位名稱 描述
point_id 回收點唯一識別碼 (Primary Key)
name 回收點名稱
address 回收點地址
location 回收點地理位置 (可用來做地圖定位)
created_at 回收點創建時間

4. `service_slots` 表格

欄位名稱 描述
slot_id 服務時段唯一識別碼 (Primary Key)
point_id 回收點 ID (Foreign Key, 來自 `recycling_points`)
date 服務日期
start_time 服務開始時間
end_time 服務結束時間
available_slots 可用名額數量

5. `delivery_schedules` 表格

欄位名稱 描述
schedule_id 排程唯一識別碼 (Primary Key)
request_id 回收申請 ID (Foreign Key, 來自 `furniture_requests`)
vehicle_id 車輛 ID (Foreign Key, 來自 `vehicles`)
eta 預計到達時間（ETA）
status 排程狀態 (如 PENDING, IN_PROGRESS, COMPLETED 等)

6. `vehicles` 表格

欄位名稱 描述
vehicle_id 車輛唯一識別碼 (Primary Key)
plate_number 車輛車牌號碼
capacity_length 車輛容量長度
capacity_width 車輛容量寬度
capacity_height 車輛容量高度
status 車輛狀態 (如 AVAILABLE, IN_USE, MAINTENANCE 等)

7. `reports` 表格

欄位名稱 描述
report_id 報表唯一識別碼 (Primary Key)
report_data 報表內容（通常為 JSON 格式）
created_at 報表創建時間
